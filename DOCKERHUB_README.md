# 国内油价监控推送工具（多用户版）

> **部署在 VPS 上，打开浏览器就能注册、配置、收推送。支持多用户，每人独立关注省份。**

## 快速启动

```bash
docker run -d \
  --name oil-price-monitor \
  --restart unless-stopped \
  -p 8080:8080 \
  -v /opt/oil-price-monitor/data:/app/data \
  -e SECRET_KEY=your-random-key \
  mrchengshi/oil-price-monitor:latest
```

然后打开浏览器 `http://VPS_IP:8080` → 注册 → 登录 → 配置即可。

## 功能

- 多用户注册 / 登录，每人独立配置
- **Web 上实时显示关注城市的当前油价**
- 自动抓取全国 31 省油价（92/95/98 汽油 + 0 号柴油）
- 油价变动时**自动推送**：邮件 / Bark（iPhone）
- 有变动才推送，不骚扰
- 浏览器一站式配置，无需改文件

## 环境变量

| 变量 | 必填 | 说明 |
|------|------|------|
| `SECRET_KEY` | 是 | 建议改成随机字符串，用于加密用户 session |

## 端口

| 端口 | 说明 |
|------|------|
| `8080` | Web 配置面板（注册 / 登录 / 配置） |

## 数据持久化

油价记录、用户配置、数据库都保存在挂载的 `/app/data` 目录下，请确保通过 `-v` 映射到宿主机，否则容器删除后数据丢失。

## 支持推送方式

| 方式 | 说明 |
|------|------|
| 邮件（SMTP） | QQ邮箱、163等，需开启SMTP并获取授权码 |
| Bark（iPhone） | App Store 下载 Bark，复制设备Key填入即可 |

## 配置项

在浏览器配置面板里可以设置：关注省份、检查间隔、邮件/Bark推送开关及详细参数。

## 可用省份

北京、上海、天津、重庆、河北、山西、辽宁、吉林、黑龙江、江苏、浙江、安徽、福建、江西、山东、河南、湖北、湖南、广东、广西、海南、四川、贵州、云南、西藏、陕西、甘肃、青海、宁夏、新疆、内蒙古
